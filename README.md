# rui-vue

## Project setup
```
yarn install
```

## Add CSS Class
```
使用本框架前，请在 CSS 中开启 border-box

*,*::before,*::after{box-sizing: border-box;}

IE 8 及以上浏览器都支持此样式
```


### Compiles and hot-reloads for development

yarn run serve
